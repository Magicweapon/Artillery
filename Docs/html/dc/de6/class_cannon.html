Detailed Description

This class deals with the behaviour of the cannon object via code in order for the player to play the game.

It contains CannonballPrefab the cannonball prefab to use it as a projectile cannonTip the child object of the cannon whose coordinates are used for spawning the cannonballs rotation private variable to control the rotation of the cannon with the player's input shotSound object used to play a shot audio clip shotSource reference to the shot audio clip used by shotSound object cannonControls reference to the CannonControls input action map to deal with the player's input aim input action variable used in conjunction with "rotation" variable to deal with the cannon's rotation shoot input action variable used to trigger cannon shot with specific player's input modifyForce input action variable to be able to increase or decrease the force used in the next shot
